This repository includes all the code used to replicate any analyis (Analysis.R) and plots (Plots.R) as well as the morphology and eco-group data (Just_spinose_foram_data.csv)

The Triton Dataset used is available at https://doi.org/10.1038/s41597-021-00942-7 and https://springernature.figshare.com/articles/dataset/Triton/13517414?backTo=%2Fcollections%2FTriton_a_new_database_of_Cenozoic_Planktonic_Foraminifera%2F5242154&file=51863741

The R version used is Version 4.5.2
